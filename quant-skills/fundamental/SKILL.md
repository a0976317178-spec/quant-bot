# 📋 基本面分析技能（Fundamental）
## 月營收評分（滿分 50 分）
| 條件 | 加分 |
|------|------|
| 月營收 YoY > 50% | +30 |
| 月營收 YoY 30~50% | +25 |
| 月營收 YoY 15~30% | +20 |
| 月營收 YoY 5~15% | +12 |
| 連續3個月YoY為正 | +10 額外 |
| 連續6個月YoY為正 | +20 額外 |
## 扣分條件
| 條件 | 扣分 |
|------|------|
| 月營收 YoY < -10% | -20 |
| 連續3個月衰退 | -25 |
| 連續6個月衰退 | -35 |
## 發布時間
每月 10 號公告上月營收，10 號前資料尚未公布。
## 進場基本面條件
1. 月營收 YoY >= 0%
2. 最近3個月趨勢向上
3. 非連續衰退中的短暫反彈
