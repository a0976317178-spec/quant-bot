# 🛡️ 風險控管技能（Risk Control）
## 核心原則
任何單筆交易最大虧損不超過總資金的 2%。
## 停損設定規則
| 股票特性 | 建議停損 |
|---------|---------|
| 大型績優股（2330等） | 5% |
| 中型成長股 | 6~7% |
| 小型高波動股 | 8~10% |
## 停利設定規則
| 策略類型 | 建議停利 |
|---------|---------|
| 短線（3~5天） | 5~8% |
| 波段（1~3週） | 10~15% |
| 中線（1~2個月） | 15~25% |
## 資金分配原則
- 單筆最大投入：15%
- 最多持股數：5 支
- 保留現金：25%
## 高風險警示
1. VIX 突破 20 → 倉位降至 50%
2. VIX 突破 25 → 倉位降至 20%
3. 單日虧損超過總資金 2% → 停止當天所有交易
4. 連續 3 筆虧損 → 停止交易，檢討策略
