# 🕯️ K線型態識別技能（Pattern Recognition）
## 買進型態（底部反轉）
### 🔥 最強型態
W底（雙底）：兩次測試低點，突破頸線量能放大 → +25
頭肩底：左肩→頭部（最低）→右肩，突破頸線確認 → +20
### 次強型態
錘子線：下影線 >= 2倍實體，出現在下跌末端 → +15
吞噬（Bullish Engulfing）：陽線完全包住前陰線，量能放大 → +12
早晨之星：大陰線→十字星→大陽線 → +18
## 賣出型態（頂部訊號）
M頭（雙頂）：兩次到達相近高點後回落 → -25
射擊之星：上影線 >= 2倍實體，出現在上漲末端 → -15
## 突破型態（最強進場信號）
箱型突破：盤整10天後向上突破，量能 >= 1.5倍 → +20
缺口突破：開盤跳空，低點不回填缺口 → +15
## 型態有效性加分
- 量能配合（放量確認）→ +10
- 在重要支撐位發生 → +5
- 同時有籌碼配合 → +10
