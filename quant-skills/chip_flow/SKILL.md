# 🏦 籌碼流向技能（Chip Flow）
## 核心邏輯
「跟著錢走」— 外資與投信同時大買，法人共識形成，上漲機率顯著提高。
## 外資評分（滿分 40 分）
| 條件 | 加分 |
|------|------|
| 外資連買 >= 5 天 | +25 |
| 外資連買 3~4 天 | +15 |
| 外資連買 1~2 天 | +8 |
| 外資5日淨買超 > 5000張 | +15 |
| 外資5日淨買超 2000~5000張 | +8 |
## 投信評分（滿分 35 分）
| 條件 | 加分 |
|------|------|
| 投信連買 >= 5 天 | +20 |
| 投信連買 3~4 天 | +12 |
| 投信 + 外資同時買超 | +15 額外加分 |
## 最強信號：三法人共買
外資 + 投信 + 自營商同日買超 → 基礎加 30 分
## 警示條件（扣分）
| 條件 | 扣分 |
|------|------|
| 外資連賣 >= 3 天 | -20 |
| 投信連賣 >= 3 天 | -15 |
| 融資大增（散戶追高） | -10 |
## 進場條件
1. 籌碼評分 >= 55
2. 外資或投信至少一方連買 3 天以上
3. 無大量融資追高現象
