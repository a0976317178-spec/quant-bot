# 📈 動能策略技能（Momentum）
## 核心邏輯
「強者恆強」— 近期表現最好的股票，未來一段時間繼續跑贏的機率高於平均。
## 評分規則
### 價格動能（滿分 40 分）
| 條件 | 加分 |
|------|------|
| 近20日報酬 > 10% | +15 |
| 近20日報酬 5~10% | +10 |
| 近60日報酬 > 20% | +15 |
| 近60日報酬 10~20% | +10 |
| 創近60日新高 | +10 |
### 均線多頭排列（滿分 30 分）
| 條件 | 加分 |
|------|------|
| MA5 > MA20 > MA60 | +20 |
| MA5 > MA20（短多） | +10 |
| 股價 > MA20 > MA60 | +10 |
### 量價配合（滿分 30 分）
| 條件 | 加分 |
|------|------|
| 突破時量能 > 1.5x 均量 | +20 |
| 量能 1~1.5x 均量 | +10 |
| 成交量持續放大（5日） | +10 |
## 進場條件
1. 動能評分 >= 60
2. 股價站在 MA20 之上
3. 今日量能 >= 1.2 倍均量
4. 大盤在季線（MA60）之上
## 停損停利建議
- 停損：跌破 MA20 或 -5%
- 停利：+12% 或出現量縮收黑
